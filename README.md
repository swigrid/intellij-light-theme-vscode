# IntelliJ IDEA Light Theme (Custom)

A custom IntelliJ IDEA Light theme for Visual Studio Code with enhanced Ruby syntax highlighting and optimized color scheme.

## Features

- Clean and professional IntelliJ IDEA-inspired light theme
- Optimized syntax highlighting for Ruby on Rails development
- Enhanced readability with carefully selected colors
- Consistent UI elements matching IntelliJ IDEA design language

## Installation

### From VS Code Marketplace

1. Open VS Code
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "IntelliJ IDEA Light Theme (Custom)"
4. Click Install
5. Press Cmd+K Cmd+T and select "IntelliJ IDEA Light (Custom)"

### Manual Installation

1. Download the latest release
2. Extract to `~/.vscode/extensions/`
3. Reload VS Code
4. Select the theme from Color Theme picker

## Customization

You can customize the theme in your VS Code settings:

```json
{
  "workbench.colorCustomizations": {
    "[IntelliJ IDEA Light (Custom)]": {
      "editor.background": "#ffffff"
    }
  }
}
```
